# IProj

A simple content management website powered by django.

[Demo](http://120.24.77.112/SignIn)

## 关于

* 前端 : Bootstrap & Jquery & Echarts
* 后台 : Django1.10 (Python3)
* 部署 : Ubuntu14.04 + Nginx + uwsgi (Aliyun)
